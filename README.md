## Features
- **Modern Layout**: Fixed detail pages, better responsiveness, and flex-based layouts.
- **Clean Aesthetic**: Inspired by modern streaming services with a dark, premium look.
- **Jellyfin 10.11+ Ready**: Updated selectors and logic to work with the latest Jellyfin web client.
- **Open Source Friendly**: Removed all proprietary assets (fonts, logos) for safe hosting. Uses "Inter" and system fonts.
- **Mobile Optimized**: Great experience on phones, tablets, and desktops.

## Usage
To use this skin, go to **Dashboard** -> **General** -> **Custom CSS** and paste the following line:

```css
@import url("https://raw.githubusercontent.com/YOUR_USERNAME/jellyflix-main/main/default.css");
```

*(Note: Replace `YOUR_USERNAME/jellyflix-main` with your actual GitHub path after uploading)*
